よくある質問

  • よくある質問

お客様より当事務所によく寄せられるご質問をまとめました。
こちらに無いご質問やご不明な点がございましたら「お問い合わせフォーム」よりお気軽にお問い合わせください。

  • Q

    まずは電話で相談をしたいのですが可能ですか？

    ANSWER

    相談は原則的には直接お会いして行っておりますが、平日の日中の面談が難しい方には、お電話でのご相談も受付けております。しかし、資料などを使用できないため対応できる内容に限りがありますので、直接お会いしての面談をおすすめします。
    初回相談は無料にて承っております。どんなご相談でも結構です。
    また、相談したら契約しなければならないということではありませんので、信頼できる税理士か見極めてみようという場合でも、一度お電話いただければ幸いです。
    なお、相続に関するご相談は匿名では受け付けできませんので、その点ご了承ください。

  • Q

    申告期限が間近でも対応してもらえますか？

    ANSWER

    はい可能です。当事務所では申告期限間近の案件や、期限後の申告も対応可能です。
    自身での申告を考えていたが、思いのほか申告書作成が難しく時間が経過してしまった。遺産分割が上手くまとまらず長引いてしまったなど。このような理由で期限が迫ってしまうことはよくあるケースです。
    当事務所では、申告期限までに遺産分割がまとまらない場合の未分割申告やその後の対応なども含めて柔軟に対応しています。

  • Q

    相続税の納税額について概算で良いので教えてもらえますか？

    ANSWER

    当サイトに簡易的なシミュレーションを用意してあります。該当ページにて必要な項目に入力すると納税額が算出されますので、ご利用ください。詳しくは、相続税シミュレーションページをご覧ください。

  • Q

    事務所への訪問はおおよそ何回位必要なのでしょうか？

    ANSWER

    初回ご面談、財産評価のご説明、最終押印時の3回程度が標準となります。
    ただし、特段お客様の方からご相談事項等が多い場合は回数が多くなる場合もあります。また、相続税申告業務は、初回のご面談と最終の署名捺印時以外は、郵送やお電話、メール等で連絡をとることも可能です。ご自宅へ訪問させて頂くことも可能ですので、お気軽にご相談ください。

  • Q

    初回相談に伺う際には何を持参すればよいのでしょうか？

    ANSWER

    ご相談に来ていただく前にお電話にてご案内させて頂きます。
    家族構成、財産内容がわかる資料を持参して頂けますとより具体的なお話しができます。

  • Q

    相続開始からいつぐらいに相談に伺えばいいのでしょうか？

    ANSWER

    出来るだけ早くご相談に来ていただけるのがベストです。
    実際には四十九日を終えてからのご相談が多いですが、相続税の申告期限が、お亡くなりになられた日から10ヶ月以内となりますので、あまり時間を置かずに早めのご相談をおすすめします。

  • Q

    申告期限内に間に合わなかった場合、どんなペナルティがあるのでしょうか？

    ANSWER

    納税期限内に税金を支払うことができなかった場合は、本来支払う相続税に加えて、「延滞税」という利息のようなものがつきます。
    また、申告期限内に申告書を提出しなかった場合は、本来の税金に加えて「無申告加算税」と言う税金も課されます。こういったことから、期限内に現金で一括で納付することが、結果的には一番支払うお金が少なくてすむことになります。

  • Q

    相続税申告と併せて、不動産登記や土地の売却もお願いしたいけど可能ですか？

    ANSWER

    はい可能です。当事務所では他士業や他関係業種とのネットワークありますので、一貫して対応させて頂くことが可能です。

  • Q

    相続人間の調停もやってもらえるのでしょうか？

    ANSWER

    いえ、当事務所では相続人の方々の間での遺産分割の調停は行っておりません。遺産分割の調停は弁護士の職域になりますので、当事務所の他士業ネットワークから相続に強い弁護士のご紹介はできます。

  • Q

    他で作成してもらった相続税申告書をチェックしてもらう事はできますか？

    ANSWER

    はい可能です。費用に関しては、前もってご提示いたします。

  • Q

    申告後に税務署から税務調査があった場合、対応してもらえますか？

    ANSWER

    はい可能です。税務調査が行われる場合、事前リハーサル～立会～修正申告まで承っております。
